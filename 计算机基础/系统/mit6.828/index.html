      # 2 Jos实验

      # 2.1 Lab1

      # 介绍

      本实验分为三个部分：

      1. 熟悉x86汇编语言、QEMU x86模拟器以及PC的开机引导过程。
      2. 检查6.828内核的引导加载程序，其位于lab的boot下。
      3. 深入研究6.828内核本身的初始模板，名为JOS，位于kernel目录下。

      # 第一部分 PC引导

      需确保环境搭建完成。

      cd ~
mkdir mit6.828
cd mit6.828
git clone https://pdos.csail.mit.edu/6.828/2018/jos.git lab
cd lab
make
make qemu
# 或 make qemu-nox

      由于我的环境是用ssh连接到的阿里云服务器，只支持help和kerninfo命令，想退出qemu只能在另一终端kill qemu-system-i38进程。
